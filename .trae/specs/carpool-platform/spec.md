# 拼车平台 - 产品需求文档

## Overview
- **Summary**: 开发一个拼车平台，包含用户注册登录、行程发布、评论功能、管理后台和首页数据展示等核心功能。
- **Purpose**: 解决用户长途出行的拼车需求，提供一个安全、便捷的拼车平台。
- **Target Users**: 有长途出行需求的用户，包括车主和乘客。

## Goals
- 实现用户注册与登录功能，确保用户必须登录才能发布行程和评论
- 优化评论功能，确保每条评论都显示对应的用户名
- 创建管理后台系统，用于监控和管理网站运行数据
- 优化首页数据展示，确保数据来源于真实业务数据
- 完成代码版本控制，将完整源代码上传至 GitHub

## Non-Goals (Out of Scope)
- 支付功能
- 实时聊天功能
- 地理位置服务
- 移动端应用开发

## Background & Context
- 项目基于 Spring Boot 2.7.15 和 Java 8 开发
- 使用 MySQL
